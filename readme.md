## How to use 
Untuk menjalankan Program ini, ikuti langkah-langkah berikut:
* Download/Clone repository ini
* Buka folder ini di dalam text editor kesayangan anda untuk melihat code nya
* masukkan link localhost/fakerPHP/contoh
* click input data dummy
* untuk mengedit data yang ditambahkan, edit di folder contoh file tambah.php
